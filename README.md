# Netflix Clone

A fully functional Netflix clone built with React and Vite.

**Created by: Kanishk Dixit**

## Features

✅ Browse movies and TV shows by category
✅ Trending content banner
✅ Search functionality
✅ Movie detail pages with trailers
✅ Responsive design
✅ Real movie data from TMDB API
✅ Smooth animations and hover effects
✅ Netflix-style UI

## Tech Stack

- React 18
- Vite
- React Router
- Axios
- TMDB API
- CSS3

## Installation

1. Navigate to the project folder:
```bash
cd netflix-clone
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit: `http://localhost:5173`

## Build for Production

```bash
npm run build
```

## Features Breakdown

- **Home Page**: Featured banner + multiple category rows
- **Movie Details**: Full movie info, ratings, genres, and trailers
- **Search**: Find any movie or TV show
- **Responsive**: Works on all screen sizes

## API

This project uses The Movie Database (TMDB) API for fetching movie data.
